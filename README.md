# cursoJS
Simulador de prestamos, para utilizarlo es necesario registrarse.
El registro de usuarios se guarda mediante localstorage, usuario y pass a eleccion del usuario.
Los prestamos tienen una tasa de interes fija.
Se pueden borrar los creditos uno a uno o borrar todos juntos.
Se pueden filtrar los prestamos buscando el monto solicitado.
Los asesores de prestamos son datos ficticios obtenidos de una api externa.
